# UncensoredJuris

UncensoredJuris is a civic discussion and debate platform designed for respectful, evidence-based conversations about law, public policy, constitutional rights, and current events.

## Purpose

The goal of this project is to create a public space where users can share ideas, debate policies, and engage in structured civic discussion while maintaining professionalism and respect.

## Features (in development)

- Public forum discussions
- Structured debate system
- Policy proposal sharing
- Lawyer directory
- Search and categories
- User profiles

## Status

Currently in early development.
